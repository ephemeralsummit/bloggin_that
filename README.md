# Web App Blog – Midterms Project

A **simple blog web application** built using **CodeIgniter 4**.

This project demonstrates basic web application development concepts using PHP and the MVC architecture. The application allows users to create, view, edit, and delete blog posts.

Repository:  
https://github.com/ephemeralsummit/webAppBlog_midterms

---

## Features

- View blog posts
- Create new posts
- Edit existing posts
- Delete posts
- Basic CRUD (Create, Read, Update, Delete) functionality
- Simple and clean UI using Bootstrap
- MySQL database integration

> ⚠️ This project is intended for **educational purposes only** and is not production-ready.

---

## Tech Stack

- **PHP 8+**
- **CodeIgniter 4**  
  https://codeigniter.com/
- **MySQL / MariaDB**
- **Bootstrap**  
  https://getbootstrap.com/
- **Composer**
